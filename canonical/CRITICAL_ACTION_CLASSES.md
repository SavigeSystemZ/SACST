# SACST Critical Action Classes

- Bootloader and boot configuration
- Initramfs and kernel changes
- Disk, partition, filesystem, and mount changes
- Encryption and key-management changes
- Firewall and routing changes
- SSH, PAM, and authentication changes
- Core service enablement or disablement
- GPU driver and compute stack changes
- Managed-switch configuration changes
- OPNsense configuration changes affecting management or traffic flow
- Firmware, factory-reset, identity, and destructive deletion changes
- Unmodeled vendor/device mutations without a prior project platform plan
