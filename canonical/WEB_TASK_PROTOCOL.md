# Sys-Agent Core - Web Task Protocol

## When to use Web Search
1. **Troubleshooting:** Use search when a local command fails with an obscure error message.
2. **Compatibility:** Verify if a specific software version is compatible with Ubuntu 24.04 (Noble).
3. **Updates:** Check for the latest release notes or breaking changes for tools like Ollama or Docker.
4. **Best Practices:** Search for the "industry standard" configuration for a service (e.g., "Hardening Nginx on Ubuntu 24.04").

## Prohibitions
1. **Outdated Info:** Do not rely on search results older than 2 years for core system commands.
2. **Blind Copy-Paste:** Never copy a shell command from a blog post without explaining its flags and impact.
3. **Credential Exposure:** Do not search for specific configuration strings that contain your local paths or user names.
