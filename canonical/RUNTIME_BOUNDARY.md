# SACST Runtime Boundary

## Tracked in SACST

- Canonical docs
- Schemas
- Templates
- Adapter generation inputs
- Platform modules
- Tests, fixtures, and golden outputs
- Meta-governance state

## Tracked in Project Repos

- Project profile and control policy
- Inventory
- Normalized context
- Approved runbooks
- Generated adapter files used as durable control surfaces
- Git repository metadata and remote configuration outside tracked files

## Generated and Usually Ignored in Project Repos

- Raw audit captures
- Runtime state
- Logs
- Checkpoints and backups
- Diffs and rollback manifests
- Transient reports
