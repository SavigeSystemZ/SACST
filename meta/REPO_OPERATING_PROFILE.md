# SACST Repo Operating Profile

This repo is the canonical template and meta-system for AI-assisted infrastructure operations.

## Primary Outcomes

- Generate self-contained project repos for host and network administration.
- Provide multi-agent instruction surfaces from shared canonical truth.
- Keep template evolution supervised, testable, and reversible.

## Non-Goals

- Hosting live instance runtime state.
- Acting as a shared runtime framework that project repos depend on by relative path.
- Replacing target-repo local instructions during later repo pivots.
